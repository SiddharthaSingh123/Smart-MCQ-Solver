# Project Overview

This project aims to build an intelligent MCQ solving system using Deep Learning and Generative AI techniques.

The project explores the complete NLP pipeline starting from text preprocessing and semantic similarity analysis to transformer-based models, retrieval-augmented generation (RAG), fine-tuning, and ensemble learning.

The objective is to develop models capable of understanding questions, analyzing answer choices, retrieving relevant information, and predicting the correct answer.
